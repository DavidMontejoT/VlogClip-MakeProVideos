# Contributing to YT Clipper

Thanks for your interest in contributing! 🎉

## How to contribute

1. **Fork** the repo
2. **Create a branch** for your feature/fix: `git checkout -b feat/my-feature`
3. **Make your changes**
4. **Test** that it works: `python3 app.py`
5. **Submit a PR** with a clear description

## Code style

- Python: follow PEP 8
- JavaScript: ES6+, no framework dependencies
- CSS: BEM-like naming, dark theme variables
- Keep it simple — this is a lightweight MVP

## What to work on

Check the **Ideas for contributors** section in the README, or look for `TODO` comments in the code.

## Questions?

Open an issue or start a discussion — happy to help!
